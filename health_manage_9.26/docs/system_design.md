## 1. 范围

### 1.1 标识

本文档适用于长远航作业应激神经系统功能预警评估系统,版本号1.0。

### 1.2 系统概述

本系统旨在开发一套高效、准确且可靠的长远航作业应激神经系统功能预警评估模型及评估系统。系统主要功能包括:

1. 应激损伤评估:利用多模态深度学习算法,构建应激损伤预警模型,评估三种应激类型(普通应激、抑郁、焦虑),准确率≥90%。
2. 多源数据采集:支持神经、生理和量表等多源数据的采集、处理和存储。
3. 实时数据可视化:展示10+关键指标,响应时间<1秒。
4. 高效系统性能:页面操作响应时间<5秒,报告分析时间<60秒,单机每分钟处理200条神经、生理信号数据等。
5. 模块化系统架构:包含模型预测、模型管理、数据管理、结果管理、用户管理等模块。

系统具有较高的安全性和可靠性要求,需要严格遵循相关军用软件开发标准。系统的主要用户包括医疗人员、研究人员和管理人员。

### 1.3 文档概述

本文档详细描述了长远航作业应激神经系统功能预警评估系统的软件设计,包括系统架构设计、详细设计、接口设计等。文档旨在为系统开发、测试和维护提供依据。

## 3. CSCI级设计决策

### 3.1 系统架构设计决策

1. 前端使用PyQt5开发桌面应用程序,后端使用Python开发。
2. 采用模块化设计,将系统划分为用户管理、数据管理、模型管理、评估功能、结果管理等模块。
3. 使用Mysql作为数据库,存储用户信息、评估结果等结构化数据。
4. 使用文件系统存储原始数据文件和模型文件。

### 3.2 安全性设计决策

1. 实现基于角色的访问控制(RBAC),区分普通用户和管理员权限。
2. 对用户密码进行加密存储。
3. 记录关键操作日志,支持审计。
4. 实现数据备份和恢复机制。

### 3.3 性能设计决策

1. 对大量数据进行批量处理,提高处理效率。
2. 使用缓存技术减少数据库访问,提高响应速度。

### 4.1 CSCI部件

系统主要包含以下软件单元:

1. 用户管理模块(user_manage)
2. 数据管理模块(data_view)
3. 模型管理模块(model_control)
4. 评估功能模块(health_evaluate)
5. 结果管理模块(results_view)
6. 系统参数管理模块(param_control)
7. 日志管理模块(log_manage)

### 4.2 执行方案

系统主要执行流程如下:

1. 用户登录 -> 进入主界面
2. 数据上传 -> 数据预处理 -> 数据可视化
3. 选择评估模型 -> 进行评估 -> 生成评估报告
4. 查看历史评估结果
5. 管理员可进行用户管理、模型管理等操作

## 5. CSCI详细设计

### 5.1 用户管理模块(user_manage)

功能:实现用户的增删改查,权限管理等。

主要类:
- User_Manage_WindowActions: 用户管理界面类
- User: 用户数据模型类

主要方法:
- add_user(): 添加用户
- delete_button(): 删除用户
- updatePSWD_button(): 修改密码

### 5.2 数据管理模块(data_view)

功能:实现数据的上传、预处理、可视化等。

主要类:
- Data_View_WindowActions: 数据查看界面类
- Data: 数据数据模型类

主要方法:
- openfile(): 上传数据文件
- show_image(): 显示数据可视化结果
- checkbutton(): 查看数据详情

### 5.3 模型管理模块(model_control)

功能:实现模型的上传、删除、导出等。

主要类:
- model_control_Controller: 模型管理界面类
- Model: 模型数据模型类

主要方法:
- uploadModel(): 上传模型
- deleteModel(): 删除模型
- exportModel(): 导出模型

### 5.4 评估功能模块(health_evaluate)

功能:实现应激损伤评估。

主要类:
- Health_Evaluate_WindowActions: 健康评估界面类
- EegModel: 评估模型类

主要方法:
- EvaluateButton(): 进行评估
- status_model(): 调用评估模型
- waitTestRes(): 处理评估结果

### 5.5 结果管理模块(results_view)

功能:实现评估结果的查看、删除等。

主要类:
- Results_View_WindowActions: 结果查看界面类
- Result: 结果数据模型类

主要方法:
- show_table(): 显示结果列表
- checkbutton(): 查看结果详情
- deletebutton(): 删除结果
