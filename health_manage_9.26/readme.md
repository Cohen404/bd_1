# 长远航作业应激神经系统功能预警评估系统

## 项目概述

本项目旨在开发一套高效、准确且可靠的长远航作业应激神经系统功能预警评估模型及评估系统。该系统为用户提供全面的应激损伤评估和管理服务，通过先进的算法和多源数据分析，实现精确的应激损伤预警和评估。

## 主要功能

1. **应激损伤评估**：利用深度学习和类脑脉冲神经网络算法，构建应激损伤预警模型，评估三种应激类型，准确率≥90%。

2. **多源数据采集**：支持神经、生理和量表等多源数据的采集、处理和存储，兼容CSV、MAT和JSON等格式。

3. **实时数据可视化**：展示10+关键指标，响应时间<1秒。

4. **高效系统性能**：
   - 页面操作响应时间<5秒
   - 报告分析时间<60秒
   - 单机每分钟处理200条神经、生理信号数据
   - 单机存储容量≥4TB，支持5000用户
   - 单机每分钟批量存储3000条采集数据

5. **模块化系统架构**：包含模型预测、模型管理、数据管理、结果管理、用户管理、角色管理、权限管理、采集参数管理和日志管理等模块。

6. **可扩展性与可维护性**：支持通过增加模块扩展功能，结构清晰，注释代码行≥25%。

## 项目结构

本项目的主要目录和文件结构如下：

- front/: 前端相关代码目录
  - admin.py: 管理员操作和权限管理的功能实现
  - change_pwd_UI.py: 实现密码修改的用户界面
  - component.py: 包含可重用的UI组件
  - data_view.py: 数据查看界面的实现
  - health_evaluate.py: 健康评估功能的实现
  - index.py: 主页面的实现，包含应激评估、结果查看、数据查看和切换登录功能
  - init_login.py: 初始登录界面的实现，提供用户登录入口
  - log_manage.py: 处理系统日志的管理功能
  - login.py: 用户登录界面的实现，包含用户名和密码输入
  - model_control_UI.py: 模型控制界面的实现，包括模型列表和上传功能
  - param_control_UI.py: 参数控制界面的实现，用于设置和管理系统参数
  - results_view.py: 结果查看界面的实现，显示当前和历史评估结果
  - user_manage.py: 用户管理界面的实现，包括用户列表和添加用户功能
- rear/: 后端相关代码目录
  - admin_rear.py: 管理员功能的后端实现
  - change_pwd_controller.py: 密码修改功能的控制器
  - data_out.py: 数据输出和处理相关功能
  - data_pretreatment.py: 数据预处理功能
  - data_view_rear.py: 数据查看功能的后端实现
  - health_evaluate_rear.py: 健康评估功能的后端实现
  - index_rear.py: 主页面功能的后端实现
  - init_login.py: 初始登录功能的后端实现
  - login_rear.py: 登录功能的后端实现
  - log_manage_rear.py: 日志管理功能的后端实现
  - model_control_controller.py: 模型控制功能的控制器
  - param_control.py: 参数控制功能的后端实现
  - results_view_rear.py: 结果查看功能的后端实现
  - user_manage_rear.py: 用户管理功能的后端实现
- sql_model/: 数据库模型相关代码目录
  - tb_data.py: 定义数据表模型，包含数据文件信息
  - tb_model.py: 定义模型表模型，包含模型类型和路径信息
  - tb_parameters.py: 定义参数表模型，包含脑电采集参数信息
  - tb_result.py: 定义结果表模型，包含评估结果信息
  - tb_user.py: 定义用户表模型，包含用户信息和权限
  - testsql.py: 数据库查询测试脚本
- state/: 状态管理相关代码目录
  - __init__.py: 状态管理模块的初始化文件
  - operate_user.py: 用户操作状态管理，包含读取、设置管理员和普通用户状态的功能
  - current_user.txt: 存储当前用户信息的文本文件
  - user_status.txt: 存储用户状态（管理员/普通用户）的文本文件
- util/: 工具函数和数据库操作相关代码目录
  - db_util.py: 数据库连接和会话管理工具
  - init_db.py: 数据库初始化脚本，包括创建数据库和表
  - test.sql: 数据库结构和初始数据的SQL脚本

注意：项目结构可能会随着开发进程而扩展或变化。其他目录和文件可能会在开发过程中添加。
